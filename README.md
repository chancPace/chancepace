# CHANS'PACE

## 프로젝트 개요
CHANS'PACE는 공간 대여 서비스를 제공하는 플랫폼으로, 사용자가 다양한 공간을 손쉽게 검색하고 예약할 수 있는 기능을 갖추고 있습니다.

## 주요 기능
- 사용자 등록 및 로그인
- 공간 검색 및 필터링
- 예약 시스템
- 결제 기능
- 기타 기능 (추가 예정)
